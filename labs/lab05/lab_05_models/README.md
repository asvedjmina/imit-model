# Lab 05 models

Модель сети Петри для задачи обедающих философов.

Структура проекта:

- `src/DiningPhilosophers.jl` — реализация сети Петри и функций анализа;
- `scripts/` — literate-источники и генерация производных форматов;
- `docs/` — Quarto-документы, полученные из literate-скриптов;
- `notebooks/` — Jupyter notebook;
- `data/` — таблицы с результатами экспериментов;
- `plots/` — итоговые графики и анимация;
- `test/runtests.jl` — базовые тесты.
