# v6.0.0 - Лабораторная работа №6: Реализация модели SIR в подходе сетей Петри

**Версия:** 6.0.0  
**Дата:** 24 апреля 2026 г.

## Описание

Шестая лабораторная работа посвящена реализации модели SIR в аппарате сетей
Петри на Julia. В проекте собран полный воспроизводимый пайплайн: модуль
модели, вычислительные сценарии, literate-источники, выполненные Jupyter
notebooks, Quarto-отчёт и презентация.

## Цели работы

1. Реализовать сеть Петри для эпидемиологической модели `S-I-R`.
2. Построить детерминированный и стохастический симуляторы.
3. Подготовить код в literate-стиле и получить `.jl`, `.ipynb`, `.qmd`.
4. Выполнить параметрическое исследование по коэффициентам `beta` и `gamma`.
5. Сгенерировать таблицы, графики и GIF-анимацию.
6. Собрать итоговые отчёт и презентацию.

## Техническая реализация

### Стек технологий

- Julia
- DrWatson.jl
- Literate.jl
- CSV.jl
- DataFrames.jl
- Plots.jl
- Quarto

### Ключевые компоненты

- модуль `SIRPetri.jl` с реализацией сети Петри;
- детерминированный симулятор SIR-модели;
- стохастический симулятор по алгоритму Гиллеспи;
- базовый и параметрический literate-сценарии;
- два выполненных Jupyter notebook и соответствующие производные форматы.
